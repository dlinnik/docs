# Заполнить вкладку

Инструмент "Заполнить вкладку" автоматически заполняет данные выбранной вкладки (маркетплейса) с помощью нейросети: категорию, название, описание, характеристики и основные поля (габариты, вес и т.д.). Это основной способ быстрой подготовки карточек к публикации.
<br>
<br>

**Где найти инструмент?**

Перейдите в раздел "Каталог товаров" → кнопка "Инструменты" → раздел "Заполнить вкладку"
<img src="./1f.png" width="840">
<br>

**Настройки**

* _Вкладка_ – выберите маркетплейс, для которого нужно заполнить данные, например OZON
* _Этапы генерации_ – выберите, что именно должна заполнить нейросеть:
  * Автоподбор категорий
  * Заполнение названий и описаний
  * Генерация характеристик
  * Заполнение основных полей (габариты и т.д.)
* _Исходные данные_ – укажите, какие атрибуты каталога использовать как источник: все или выбранные вручную
* _Поиск недостающей информации в интернете_ – если включено, нейросеть будет дополнительно искать данные о товаре в сети
* _Генерировать только для незаполненных полей_ – нейросеть пропустит поля, в которых уже есть данные. Отключите, если хотите перегенерировать всё
<br>

**Расширенные настройки**

Позволяют задать собственный промпт для генерации названия и описания. По умолчанию используется стандартный промпт Databird, оптимизированный под требования маркетплейса.
<img src="./2f.png" width="840">
<br>

⚠️ Инструмент расходует кредиты проекта. Чем больше товаров и этапов генерации выбрано — тем больше кредитов будет потрачено<br>
⚠️ Результаты генерации рекомендуется проверить перед экспортом — нейросеть может ошибиться в категории или характеристиках, если исходные данные товара неполные
