# Включить отображение фото в каталоге товаров, если их не видно

❗При получении фотографий по ссылке с устаревшим протоколом HTTP (не HTTPS) фотографии товаров могут некорректно отображаться в каталоге Databird. 

![Untitled](Untitled%2078.png)

Следующим способом вы можете включить картинки в браузере Google Chrom:

1. Войти в настройки Chrome

![Untitled](Untitled%2079.png)

2. Конфиденциальность и безопасность => Настройки сайтов

![Untitled](Untitled%2080.png)

3. Дополнительные настройки контента

![Untitled](Untitled%2081.png)

4. Небезопасный контент - считается небезопасным, потому что на картинки на вашем сайте смотрит

![Untitled](Untitled%2082.png)

5. Добавить в качестве доверенного сайта [app.databird.ru](https://app.databird.ru/)

![Untitled](Untitled%2083.png)

✅ Теперь фотографии будут корректно отображаться в каталоге.

![Untitled](Untitled%2084.png)
