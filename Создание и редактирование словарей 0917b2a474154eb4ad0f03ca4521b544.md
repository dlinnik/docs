# Создание и редактирование словарей

Для создания нового словаря следует в левом окне сайта перейти во вкладку **“Словари”**.

![Untitled](%D0%A1%D0%BE%D0%B7%D0%B4%D0%B0%D0%BD%D0%B8%D0%B5%20%D0%B8%20%D1%80%D0%B5%D0%B4%D0%B0%D0%BA%D1%82%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D0%BD%D0%B8%D0%B5%20%D1%81%D0%BB%D0%BE%D0%B2%D0%B0%D1%80%D0%B5%D0%B9/Untitled.png)

На отрывшейся странице будут отображаться уже созданные ранее словари. За создание нового отвечает кнопка *“Добавить словарь”* в правом верхнем углу страницы. 

![Untitled](%D0%A1%D0%BE%D0%B7%D0%B4%D0%B0%D0%BD%D0%B8%D0%B5%20%D0%B8%20%D1%80%D0%B5%D0%B4%D0%B0%D0%BA%D1%82%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D0%BD%D0%B8%D0%B5%20%D1%81%D0%BB%D0%BE%D0%B2%D0%B0%D1%80%D0%B5%D0%B9/Untitled%201.png)

В открывшемся окне нужно дать словарю имя, после чего нажать на кнопку *“Сохранить”*.

![Untitled](%D0%A1%D0%BE%D0%B7%D0%B4%D0%B0%D0%BD%D0%B8%D0%B5%20%D0%B8%20%D1%80%D0%B5%D0%B4%D0%B0%D0%BA%D1%82%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D0%BD%D0%B8%D0%B5%20%D1%81%D0%BB%D0%BE%D0%B2%D0%B0%D1%80%D0%B5%D0%B9/Untitled%202.png)

После проделанных действий во вкладке **“Словари”** будет создан новый пустой шаблон, щелчком кнопки мыши по его названию, перейдем к его редактированию.

![Untitled](%D0%A1%D0%BE%D0%B7%D0%B4%D0%B0%D0%BD%D0%B8%D0%B5%20%D0%B8%20%D1%80%D0%B5%D0%B4%D0%B0%D0%BA%D1%82%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D0%BD%D0%B8%D0%B5%20%D1%81%D0%BB%D0%BE%D0%B2%D0%B0%D1%80%D0%B5%D0%B9/Untitled%203.png)

Так как каждый словарь представляет собой таблицу данных, то создать его наполнение можно в редакторе таблиц Excel.

![Untitled](%D0%A1%D0%BE%D0%B7%D0%B4%D0%B0%D0%BD%D0%B8%D0%B5%20%D0%B8%20%D1%80%D0%B5%D0%B4%D0%B0%D0%BA%D1%82%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D0%BD%D0%B8%D0%B5%20%D1%81%D0%BB%D0%BE%D0%B2%D0%B0%D1%80%D0%B5%D0%B9/Untitled%204.png)

❕Наполнение словаря создается по определенным правилам. Первая строка должна содержать в себе названия колонок. Колонки, где отсутствует первая строка, при создании словаря будут проигнорированы. Остальные ячейки в колонке не обязательны к заполнению.

Когда табличный файл с наполнением для словаря будет готов, можно загрузить его в созданный шалон, нажав на кнопку “Загрузить словарь”.

![Untitled](%D0%A1%D0%BE%D0%B7%D0%B4%D0%B0%D0%BD%D0%B8%D0%B5%20%D0%B8%20%D1%80%D0%B5%D0%B4%D0%B0%D0%BA%D1%82%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D0%BD%D0%B8%D0%B5%20%D1%81%D0%BB%D0%BE%D0%B2%D0%B0%D1%80%D0%B5%D0%B9/Untitled%205.png)

Подождите немного и вы увидите, что шаблон стал словарем.

![Untitled](%D0%A1%D0%BE%D0%B7%D0%B4%D0%B0%D0%BD%D0%B8%D0%B5%20%D0%B8%20%D1%80%D0%B5%D0%B4%D0%B0%D0%BA%D1%82%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D0%BD%D0%B8%D0%B5%20%D1%81%D0%BB%D0%BE%D0%B2%D0%B0%D1%80%D0%B5%D0%B9/Untitled%206.png)

❕Если при создании наполнения словаря вы допустили ошибку или вам требуется расширить его, просто загрузите исправленный табличный файл, воспользовавшись кнопкой *“Загрузить словарь”*.

✔️ Созданный таким образом словарь вы можете использовать при написании правил, чтобы удобно переводить данные из одного формата в другой.
