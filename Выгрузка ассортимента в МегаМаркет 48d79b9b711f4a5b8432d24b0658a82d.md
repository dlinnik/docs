# Выгрузка ассортимента в МегаМаркет

МегаМаркет, как и Яндекс Маркет, построен по принципу одной карточки для всех продавцов. Поэтому сначала выгружаются предложения ассортимента продавца в формате YML фида. Ассортимент сопоставляется с текущими карточками площадки на основе штрихкодов. Если предложение не может быть сопоставлено, то продавец должен создать новую карточку товара.

Процесс выгрузки:

1. Создание YML экспорта для генерации фида предложений.
    
    Создайте и настройте YML фид с товарными предложениями.
    

![Untitled](Untitled%20176.png)

1. Загрузка фида в МегаМаркет по ссылке
    
    Скопируйте ссылку на фид в “Настройках” экспорта YML.
    
    Вставьте ссылку в “Загрузку и обновление” в кабинете МегаМаркета.
    
    Первая загрузка ассортимента может занять около 5 часов.
    

![Untitled](Untitled%20177.png)

![Untitled](Untitled%20178.png)

1. Проверка соответствия товаров
    
    Проверьте правильность предложений.
    
    Убедитесь, что каждому предложению соответствует карточка товаров. 
    
    Подтвердите соответствия для неполного соответствия.
    

![Untitled](Untitled%20179.png)

1. Создание экспорта карточек, для ассортимента, который нельзя сопоставить

Сервис сгенерирует архив Excel файлов, по одному на каждую категорию товаров МегаМаркета.

![Untitled](Untitled%20180.png)

1. Загрузка карточек на МегаМаркет
    
    Загрузите сгенерированные Excel файлы и проверьте правильность создания карточек и их связь с ассортиментом.
