# Что такое правила?

Правило - это выражение, которое приравнивается атрибуту системы Databird. Используя логику заданного выражения, Databird будет вычленять и сопоставлять данные ваших импортов и экспортов. Правила выполняются в порядке приоритета. 

![Untitled](./Untitled.png)

❕Приоритет выполнения, изначально равен 0, чем больше это значение, тем позже будет обработано правило в сравнении с другими.

Для написания тела правила (логики) используется язык JSONata. Подробнее о часто используемых для написания правил возможностях языка JSONata можно прочитать в 6 главе Документации пользователя.
