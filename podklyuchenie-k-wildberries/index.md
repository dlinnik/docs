# 

Для подключения маркетплейса Wildberries к сервису Databird нужно заполнить обязательное поле “Токен Wildberries”

![Untitled](./Untitled.png)

🧩 Перейдите [в раздел «Интеграции по API»](https://seller.wildberries.ru/api-integrations) в профиле и нажмите «+ Создать токен». Откроется окно создания токена с двумя вкладками. Выберите вкладку в зависимости от типа токена, который вам нужен.

1. Разделе настроек “Интеграция по API” нажмите кнопку «+ Создать токен»

![Снимок экрана 2025-11-11 в 09.25.58.png](./%D0%A1%D0%BD%D0%B8%D0%BC%D0%BE%D0%BA_%D1%8D%D0%BA%D1%80%D0%B0%D0%BD%D0%B0_2025-11-11_%D0%B2_09.25.58.png)

1. В открывшемся разделе создания токена выберите вкладку “Для интеграции вручную”

![Снимок экрана 2025-11-11 в 09.24.52.png](./%D0%A1%D0%BD%D0%B8%D0%BC%D0%BE%D0%BA_%D1%8D%D0%BA%D1%80%D0%B0%D0%BD%D0%B0_2025-11-11_%D0%B2_09.24.52.png)

1. Выберите “Персональный токен”

![Снимок экрана 2025-11-11 в 09.25.49.png](./63e48239-2595-42da-9fd3-0a76d4440eb9.png)

1. Дайте название токену, например “Токен Databird”

![Снимок экрана 2025-11-11 в 09.40.13.png](./%D0%A1%D0%BD%D0%B8%D0%BC%D0%BE%D0%BA_%D1%8D%D0%BA%D1%80%D0%B0%D0%BD%D0%B0_2025-11-11_%D0%B2_09.40.13.png)

1. Выберите, с какими категориями API будет работать данный токен
    
    ✅ **Необходимый минимум для корректной работы сервиса**: Контент, Маркетплейс, Статистика, Цены и скидки
    
    ![Снимок экрана 2025-11-11 в 09.40.05.png](./%D0%A1%D0%BD%D0%B8%D0%BC%D0%BE%D0%BA_%D1%8D%D0%BA%D1%80%D0%B0%D0%BD%D0%B0_2025-11-11_%D0%B2_09.40.05.png)
    
2. Установите уровень доступа на “Чтение и запись”

![Снимок экрана 2025-11-11 в 09.40.21.png](./%D0%A1%D0%BD%D0%B8%D0%BC%D0%BE%D0%BA_%D1%8D%D0%BA%D1%80%D0%B0%D0%BD%D0%B0_2025-11-11_%D0%B2_09.40.21.png)

1. Завершите остальные настройки и нажмите кнопку **”**Создать**”**
2. Скопируйте готовый токен в поле “**Токен Wildberries**”
