**Источник данных** - это способ получения данных о товарах в различных форматах.

![Новый проект (8).png](./%25D0%259D%25D0%25BE%25D0%25B2%25D1%258B%25D0%25B9_%25D0%25BF%25D1%2580%25D0%25BE%25D0%25B5%25D0%25BA%25D1%2582_(8).png)

| **JSON** | Получение товаров из файла в формате JSON |
| --- | --- |
| **YML** | Получение товаров из файла в стандарте YML |
| **XML** | Получение товаров из файлов XML |
| **XLSX** | Получение товаров из файлов Excel таблиц |
| **GSS** | Получение товаров из файлов из Google Таблиц |
| **CSV** | Получение товаров из файлов CSV |
| **API** | Получение товаров по JSON API (настройка заголовков, параметров, пагинации) |
