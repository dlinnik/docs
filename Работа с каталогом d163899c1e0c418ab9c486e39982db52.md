# Работа с каталогом

Каталог - это хранилище данных о товарах продавца. Каталог позволяет хранить всю информацию о товаре для ее дальнейшей загрузки в маркетплейсы и другие торговые точки.

Чем более полная информация о товаре представлена в каталоге, тем больше параметров можно заполнить при экспорте.

## Как загрузить товары в каталог?

Для загрузки товаров есть 2 способа:

1. **Загрузка из источника данных**. Это способ позволяющий гибко загрузить данные из открытых форматов (xml, yml, excel, google sheet), маркетплейсов и других учетных систем. Источник может быть настроен на “регулярную” автоматическую загрузку и обновления данных о товарах, влючая цены и остатки.
2. **Обновление из простого Excel документа**. 

## Как загрузить простой файл Excel?

Документ должен включать в себя первую строчку с названиями атрибутов. Названия полей должны соответствовать атрибутам в каталоге. Обязательно наличие столбца с заголовком “ID товара”, так как именно этот заголовок позволяет установить соответствие загружаемых товаров с уже существующими товарам в каталоге. Если товар уже есть в каталоге, то его атрибуты будут обновлены, если нет, то будет создан новый товар.

## Как сохранить товары каталога в Excel?

Вы можете сократить список выгружаемых товаров с помощью фильтров каталога.

Нажмите кнопку “Сохранить в Excel” для сохранения данных о товарах в файл.

## Как массово обновить или добавить новые данные?

Для массового обновления вы можете отредактировать выгруженный файл и загрузить его обратно. В файле можно оставить только обновляемые или добавляемые колонки, но важно, чтобы присутствовала колонка “ID товара”.

## Как отредактировать единичный товар?

Нажмите на ссылу “ID товара” для перехода в карточку товара. Вы можете изменить значения основных атрибутов или добавить новые характеристики товара. Товар может иметь неограниченное количество атрибутов характеристик.
