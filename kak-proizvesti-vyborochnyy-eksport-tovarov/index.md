# Как произвести выборочный экспорт товаров

Частичный (выборочный) экспорт производится при помощи фильтров. 

📎 Подробнее о фильтрах 

[Настройка фильтров](%D0%9D%D0%B0%D1%81%D1%82%D1%80%D0%BE%D0%B9%D0%BA%D0%B0%20%D1%84%D0%B8%D0%BB%D1%8C%D1%82%D1%80%D0%BE%D0%B2%2067018db9aca848158452fe98eb821a4f.md)

### Экспорт определенных товаров (по ID товара)

Для этого заходим во вкладку “Фильтр товаров” и добавляем новый фильтр.

![Untitled](./Untitled.png)

В первом поле из выпадающего списка выбираем “ID товара”, во втором - “в списке”, а в третьем через запятую вводим ID товаров (из каталога Databird) подающихся на экспорт.

![Untitled](./Untitled%201.png)

Сохраняем результат. И наблюдаем, что количество выбранных для экспорта товаров уменьшилось. Нажатием по кнопке “Открыть в каталоге” можно посмотреть какие именно товары были вами выбраны.

![Untitled](./Untitled%202.png)

![Untitled](./Untitled%203.png)

Произведем пробную выгрузку. Жмем кнопку “Проверка” и наблюдаем наши товары во вкладке “Данные”.

![Untitled](./Untitled%204.png)

### Экспорт товаров по атрибутам (категория, размер, производитель и т.д…)

Данная выгрузка тоже производится с заранее заданным фильтром. Однако теперь в первом поле из выпадающего списка выбираем интересующий нас атрибут, а последнем прописываем интересующие нас значения.

![Untitled](./Untitled%205.png)

Не забываем сохранить фильтр и провести тестовую выгрузку.
