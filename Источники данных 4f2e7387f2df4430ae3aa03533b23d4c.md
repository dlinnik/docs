# Источники данных

**Источник данных** - это способ получения данных о товарах в различных форматах.

Источник позволяет разово или регулярно загружать данные о новых товарах, обновлять данные об уже существующих, а так же проверять данные на наличие ошибок или несоответствий. Различные настройки источника позволяют комбинировать вариант только загрузки новых товаров, обновления конкретных полей уже существующих товаров, а так же  модифицировать данные при загрузке в зависимости от нужной бизнес логики продавца.

## Создание источника

Для создания нового источника следует в левом окне сайта выбрать вкладку “Источники данных**”**.

![Untitled](%D0%98%D1%81%D1%82%D0%BE%D1%87%D0%BD%D0%B8%D0%BA%D0%B8%20%D0%B4%D0%B0%D0%BD%D0%BD%D1%8B%D1%85/Untitled.png)

На отрывшейся странице будут отображаться уже созданные ранее источники данных. За  создание нового подключения отвечает кнопка *“Добавить подключение”* в правом верхнем углу страницы. 

![Untitled](%D0%98%D1%81%D1%82%D0%BE%D1%87%D0%BD%D0%B8%D0%BA%D0%B8%20%D0%B4%D0%B0%D0%BD%D0%BD%D1%8B%D1%85/Untitled%201.png)

Выберите тип и перейдите к созданию нового и источника:

![Untitled](%D0%98%D1%81%D1%82%D0%BE%D1%87%D0%BD%D0%B8%D0%BA%D0%B8%20%D0%B4%D0%B0%D0%BD%D0%BD%D1%8B%D1%85/Untitled%202.png)

Подробнее о типах источников: 

[Типы источников](%D0%A2%D0%B8%D0%BF%D1%8B%20%D0%B8%D1%81%D1%82%D0%BE%D1%87%D0%BD%D0%B8%D0%BA%D0%BE%D0%B2%20b5e2bee0a2fc4c1d9f472d9bf88d115e.md)

В открывшемся окне *“Создание нового источника”*  заполните все обязательные поля в зависимости от типа источника и завершите создание подключение нажав по соответствующей кнопке.

![Untitled](%D0%98%D1%81%D1%82%D0%BE%D1%87%D0%BD%D0%B8%D0%BA%D0%B8%20%D0%B4%D0%B0%D0%BD%D0%BD%D1%8B%D1%85/Untitled%203.png)

Дождитесь окончания проверки источника. Если проверка прошла успешно - можно приступать к дальнейшей настройке.

![Untitled](%D0%98%D1%81%D1%82%D0%BE%D1%87%D0%BD%D0%B8%D0%BA%D0%B8%20%D0%B4%D0%B0%D0%BD%D0%BD%D1%8B%D1%85/Untitled%204.png)

Во вкладке “Товары” будут отображены все данные по товарным единицам, полученным из источника. Вы можете нажать кнопку “Добавить в каталог”, чтобы все товары из фида сразу попали в ваш каталог Databird, или можете воспользоваться фильтром-поиском, выбрав необходимые для загрузки в каталог позиции.

![image.png](%D0%98%D1%81%D1%82%D0%BE%D1%87%D0%BD%D0%B8%D0%BA%D0%B8%20%D0%B4%D0%B0%D0%BD%D0%BD%D1%8B%D1%85/image.png)

После нажатия по кнопке “Добавить в каталог”, появится окно, где нужно будет подтвердить свой выбор.

![image.png](%D0%98%D1%81%D1%82%D0%BE%D1%87%D0%BD%D0%B8%D0%BA%D0%B8%20%D0%B4%D0%B0%D0%BD%D0%BD%D1%8B%D1%85/image%201.png)
