# Установка периода загрузки и выгрузки

Чтобы каждый раз вручную не совершать импорт и экспорт данных с уже готовыми правилами, можно установить период загрузки и выгрузки. В обоих случаях это делается во владке **“Настройки”**.

![Untitled](Untitled%2052.png)

По умолчанию период загрузки установлен “вручную”, это означает, что работа с данными будет производиться только по вашему нажатию на кнопку *“Выполнить”*.

![Untitled](Untitled%2053.png)

Если вы хотите автоматизировать этот процесс, то вместо пункта “вручную” следует выбрать другой интересующий вас вариант. 

![Untitled](Untitled%2054.png)

Далее нужно установить время таймера, после чего нажать кнопку “ОК”.

![Untitled](Untitled%2055.png)

- Если **период загрузки “Таймер”** - то спустя установленное вами время произойдет автоматическое выполнение загрузки/выгрузки.
- Если **период загрузки выбран другой** - то в установленное вами время произойдет автоматическое выполнение загрузки/выгрузки.

При еженедельном периоде загрузки можно дополнительно указать в какие именно дни производить загрузку/выгрузку.

![image.png](image%209.png)

 Сохранить результат можно щелчком по соответствующей кнопке.

![Untitled](Untitled%2056.png)

Отлично, теперь загрузка и выгрузка будут происходить автоматически, без вашего непосредственного участия.
