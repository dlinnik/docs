# Настройки обновления каталога

Настройки обновления каталога находятся внизу вкладки “Настройки” каждого отдельного подключения.

- **Создание новых товаров** - в зависимости от выбора, в момент импорта, создает или нет товар в каталоге Databird при его отсутствии в нем.
- **Добавление исходных полей** - в зависимости от выбора, в момент импорта, добавляет или нет товарам в каталоге Databird новые атрибуты, если их ранее не было.
- **Список атрибутов для обновления существующих товаров** - добавьте в этот список те атрибуты, которые должны будут обновиться у товаров, пришедших в каталог Databird из этого подключения. Остальные атрибуты обновлены не будут.
- **Список атрибутов для обновления существующих товаров -** если товар отсутствует в подключении (хотя приходил из него), но присутствует в каталоге Databird, то атрибуты из указанного списка будут удалены у данного товара в каталоге. Остальные атрибуты удалены не будут.
