# Создание копии источника с сохранением настроек атрибутов

Для создания копии необходимо перейти в уже существующий источник данных и нажать на многоточие рядом с кнопкой “Загрузить”

![Untitled](./Untitled.png)

Далее в выпадающего списке найти пункт “Конфигурация” и нажать по нему

![Untitled](./Untitled%201.png)

Через некоторое время (в зависимости от объема настроек в источнике) на ваш ПК будет скачан конфигурационный файл 

![Untitled](./Untitled%202.png)

Теперь можно перейти во вкладку “Источники данных” и добавить новый источник, нажав по соответствующей кнопке

![Untitled](./Untitled%203.png)

❗Обратите внимание на то, что тип нового источника должен совпадать с типом источника, откуда был взят файл конфигурации (в данном случае GSS)

При создании нового источника нажимаем на многоточие в правом верхнем углу 

![Untitled](./Untitled%204.png)

Из выпадающего списка выбираем “Загрузить конфигурацию” и в открывшемся далее окне проводника выбираем недавно скачанный файл конфигурации

![Untitled](./Untitled%205.png)

Копия источника готова, она появится в списке с источниками данных спустя некоторое время

![Untitled](./Untitled%206.png)

❗Если вы создаете копии в пределах одного проекта, то советуем переименовать клонированные источники, дабы избежать ошибок при дальнейшей работе (Пункт “Переименовать” в выпадающем списке, появляющемся при нажатии по многоточию в правом верхнем углу)

![Untitled](./Untitled%207.png)
