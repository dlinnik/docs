# Типы экспортов

**Экспорт данных** - это способ отправки данных о товарах в различных форматах на интересующие вас площадки.

| **OZON_CARDS** | Выгрузка карточек товаров на маркетплейс OZON |
| --- | --- |
| **OZON_STOCKS** | Выгрузка остатков товаров на маркетплейс OZON |
| **OZON_PRICES** | Выгрузка цен товаров на маркетплейс OZON |
| OZON_ACTIONS | Управление вхождением товаров в акции OZON |

# Энричеры (Обогатители контента)

| **IMAGE_RESIZER** | Изменение размера изображений под заданные значения (возвращает XLSX файл) |
| --- | --- |
| **AI_GENERATOR** | Сгенерировать поля экспорта для конкретного маркетплейса на основе данных из названия и описания товара (возвращает XLSX файл) |
| **AI_SEARCH** | Поиск данных о товарах в интернете (возвращает XLSX файл) |
